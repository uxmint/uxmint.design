CHILD CARE PORTAL

A worldwide NGO approached us for thier PoC on tablet based application for children in under-previlidged countries. The application would be used on shared tablets by multiple children.

ABOUT PROJECT

Application for children in under-previlidged countries, to be used under guidance of the NGO for learning, interaction with sponsor and community.

  • PLATFORM

    Android App

  • SEGMENT

    Education

SOLUTION

- Easy login with facial recognition

- Simple and Enjoyable interactive games for learning

- Bonding with the sponsor

- Different solutions for diffent age band

- Remote data loading to allow usage even in regions of no network connectivity

Our solutions were to enable children using the app intrinsic motivation and without any difficulties
