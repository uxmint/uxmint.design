CASE STUDY

APP DEVELOPMENT

Platform for designers, design aspirants/students, mentor to collaborate and share their ideology, design method, process and uplift their skills in the Design domain.

WHAT WE DID

  • UX Research
  • UI Design
  • Android & iOS
  • Back End Development

ABOUT PROJECT

The user base was scattered across multiple segments of "Students, Mentors, Designers" the requirement was to service every segment of the user differently. Some were using the application to only browse through or consume the information while others were active participants who contributed to the content. The framework had to be chosen carefully to cater all the needs seemlessly.

PLATFORM WE USED

Native platforms were chosen based on the User Research to cover all the user segments

  • Android

  • iOS

FRAMEWORK WE USED

We had chosen React Native & ReactJS with Microservices (Spring Boot) to service all the UI interfaces.

  • React JS
